## Usage

这是一个简单的密码管理器，用于存储密码和其他敏感信息。您可以使用它执行以下操作：

- **复制密码**：将所需的密码复制到剪贴板
- **新密码**：生成一个强密码并将其保存到加密存储中
- **导入密码**：从外部加密存储中导入密码

该脚本只是在 `pass` 之上的 GUI 封装，`pass` 是一款流行的命令行密码管理器：https://www.passwordstore.org/
