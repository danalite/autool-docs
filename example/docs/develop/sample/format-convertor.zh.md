## 使用方法

只需运行脚本，您将被提示选择要转换为的格式。为简单起见，此脚本仅实现了从 MOV 到 GIF 的转换。您可以通过修改脚本添加更多格式。在下面的 GIF 中，您可以选择要转换的源文件，转换文件的质量以及转换后的 GIF 的 FPS（每秒帧数）。

此文档中的所有 GIF 都是使用此脚本生成的。
